# Salate Time

Salate Time is a Flutter application targeting Android, iOS, web, macOS, Linux, and Windows.

This site is the project's documentation home. It is meant to grow alongside the codebase — update the relevant page in the same change that adds or alters behavior, rather than batching docs work later.

## Where to start

- [Getting Started](getting-started.md) — install dependencies and run the app locally.
- [Architecture](architecture.md) — how the codebase is organized and the reasoning behind it.
- [Contributing](contributing.md) — conventions for code and for keeping these docs current.
- [Changelog](changelog.md) — notable changes over time.

## Project status

The app is currently at the default `flutter create` template stage (a single counter screen in `lib/main.dart`). The docs below describe the project as it exists today and should be expanded as features are built out.
